Message de la ministre

C’est avec grand plaisir que je vous annonce que le Code de valeurs et d’éthique a reçu l’approbation du Comité de direction. Il est en vigueur à partir d’aujourd’hui, vous pouvez donc le consulter sur l’intranet dès maintenant.

Le Code a pour but de nous orienter dans l’affirmation des valeurs d’une fonction publique professionnelle dans toutes les facettes de notre travail. Il s’agit d’un document important avec lequel tous les employés doivent se familiariser.

La création d’un code est un grand pas en avant, mais l’établissement d’une culture organisationnelle fondée sur les valeurs et l’éthique est un travail qui doit se poursuivre en permanence. Je souhaite donc que des discussions approfondies sur les façons de concrétiser les valeurs et l’éthique aient lieu dans toutes les équipes de travail.

Notre conseiller responsable des questions de valeurs et d’éthique, Neil Gallagher, prévoit rencontrer tous les gestionnaires au cours des prochains mois afin de connaître leurs préoccupations et de répondre à leurs questions. Il compte également concevoir des outils pour aider les employés à mettre en application le Code dans leur travail au quotidien.

J’aimerais profiter de l’occasion pour remercier tous les employés qui ont participé à l’élaboration du Code ainsi que ceux qui ont pris le temps de formuler des commentaires judicieux ce qui a permis de l’améliorer. Grâce à leur travail, nous avons un code sur lequel nous pouvons nous appuyer.

Valérie Barreta
